# 💧 C-Nath Water Solution

C-Nath Water Solution is a professional RO (Reverse Osmosis) water purifier sales and service management platform. It is designed to simplify customer management, service tracking, AMC management, installation records, and spare parts history for water purifier businesses.

## 🌐 Live Website

https://cnath.in

---

# ✨ Features

- 👤 Customer Management
- 🛠️ RO Installation Records
- 🔧 Service History Tracking
- 📅 AMC (Annual Maintenance Contract) Management
- 💧 Water Purifier Sales
- 📦 Spare Parts Replacement Records
- 📍 Customer Address & Contact Management
- 🔍 Service Search & Filter
- 📱 Responsive Design
- 📞 Contact & Inquiry Form

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/cnath-water-solution.git
```

Move into project folder

```bash
cd cnath-water-solution
```

Install dependencies

```bash
npm install
```


```

Run the project

```bash
npm start
```

or

```bash
npm run dev
```
